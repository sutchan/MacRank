# 项目定义 (Project Definition)

## 项目概况
- **项目名称**: MacRank
- **当前版本**: v0.3.1
- **项目描述**: 针对 Apple Silicon (M1-M4) 及经典 Intel Mac 的交互式性能天梯榜与 AI 导购助手，支持与 PC 硬件进行性能对比。
- **GitHub**: https://github.com/sutchan/MacRank

## 核心价值主张
1. **数据透明**: 聚合 Geekbench 6 和 Metal 跑分，提供客观的性能基准，支持 PC 硬件参照对比。
2. **决策辅助**: 通过对比模式、散点图（性价比）、多场景评分和 AI 顾问，解决用户"买哪款 Mac"的决策瘫痪问题。
3. **极佳体验**: 打造原生级的 macOS UI 风格和流畅的交互体验，支持离线使用。

## 用户画像
- **开发者**: 需要了解多核编译性能和内存配置。
- **设计师/创作者**: 关注 GPU (Metal) 性能和渲染能力。
- **学生/普通用户**: 寻找性价比最高的入门机型。

## 关键成功指标 (KPIs)
- **页面加载速度 (LCP)**: < 1.5s
- **AI 响应时间**: < 3s
- **离线可用性**: PWA 支持断网访问核心榜单，AI 具备离线降级方案。
- **URL 同步**: 所有筛选状态可通过 URL 分享。

## 核心特性
- **多场景评分**: 支持 balanced (平衡)、developer (开发者)、creative (创意)、daily (日常) 四种评分模式
- **PC 参照系**: 可开启 PC/GPU 参考硬件作为性能锚点
- **离线 AI**: AI 顾问具备离线降级逻辑，网络不可用时仍能提供基础建议
- **多语言**: 支持 10 种语言界面
- **深色模式**: 支持手动/自动深色模式切换
- **分享功能**: 支持 Web Share API 和剪贴板复制链接
