# 项目路线图 (Roadmap)

## 当前版本: v0.1.18

### 已完成 (Completed)
- [x] **基础框架**: React 19 + Vite + Tailwind CSS。
- [x] **核心数据**: M1-M4 系列及 2010+ Intel 机型数据库。
- [x] **排行榜**: 综合评分算法及响应式表格，支持**移动端列冻结**。
- [x] **对比功能**: 双机型参数与性能直观对比，引入**五维雷达图**。
- [x] **AI 顾问**: 集成 Gemini 3 Flash 进行智能问答。
- [x] **国际化**: 支持 10 种主要语言，完成**模块化重构**。
- [x] **设置优化**: 重构设置菜单，优化语言选择与系统选项。
- [x] **PWA**: 支持离线缓存和安装。

---

## 短期计划 (v0.2.x) - Q2 2025

### 1. 数据增强
- [ ] **历史价格曲线**: 记录主要机型的价格波动（需接入第三方价格 API 或手动维护历史低价）。
- [ ] **更多基准测试**: 引入 Cinebench R23/2024 分数，作为 Geekbench 的补充。
- [ ] **SSD 速度**: 增加硬盘读写速度字段，特别是针对 "Base Model" 降速问题的标注。

### 2. 功能优化
- [ ] **分享功能**: 生成当前对比结果或排行榜的图片/链接，便于社交分享。
- [ ] **收藏夹**: 允许用户收藏感兴趣的机型（本地存储）。
- [ ] **高级筛选**: 增加按 "内存 >= 16GB" 或 "价格 < $1500" 的范围筛选。

### 3. UI/UX 改进
- [ ] **骨架屏 (Skeleton)**: 优化数据加载时的视觉体验。
- [ ] **图表交互增强**: 移动端图表支持手势缩放。

---

## 中期计划 (v0.5.x) - Q3 2025

### 1. 社区功能
- [ ] **用户评论**: 集成轻量级评论系统 (如 Giscus)，允许用户在机型详情页留言。
- [ ] **真实续航报告**: 收集用户提交的实际续航时间，对比官方数据。

### 2. 生态扩展
- [ ] **配件推荐**: 基于机型推荐显示器、扩展坞等配件（潜在的商业化路径）。
- [ ] **macOS 兼容性**: 标注各机型支持的最高 macOS 版本（针对 Intel 老机型）。

---

## 长期愿景 (v1.0.0)

- **全苹果生态覆盖**: 扩展至 iPhone (A系列芯片) 和 Apple Watch。
- **3D 模型预览**: 使用 Three.js 在详情页展示机型 3D 模型。
- **个性化推荐引擎**: 基于用户职业（程序员、设计师、学生）的问卷式推荐系统。
